FEATURE 1 – CONVERSATIONAL DASHBOARD UPDATES

The dashboard must support follow-up queries within the same chat session.

When the user asks a follow-up question, the system should update or modify the existing dashboard instead of creating a new one.

Example conversation:

User:
Show monthly revenue by region

AI:
Displays line chart.

User:
Now filter this to only show East region

AI:
Dashboard updates to filtered data.

User:
Compare Q2 vs Q3

AI:
Adds comparison chart.

User:
Show top 5 campaigns

AI:
Updates dashboard accordingly.

The system must maintain dashboard context during the conversation.

---

FEATURE 2 – AI INSIGHT SUMMARY

After generating a dashboard, automatically display an AI-generated insight summary.

Place this section above or beside the charts.

Example:

Insight Summary

• East region generated the highest revenue.
• Influencer campaigns perform best with Gen-Z audience.
• Revenue increased by 22% from Q2 to Q3.

This should be generated automatically from the query result.

---

FEATURE 3 – BUSINESS REPORT GENERATOR

Add a button labeled:

Generate Business Report

When clicked, the system should generate a structured business report based on the dashboard data.

Report format:

Title

Executive Summary

Key Findings

Data Highlights

Business Recommendations

Allow users to download the report as:

Markdown (.md)

The report should summarize the key insights from the charts.

FEATURE 4 – SMART QUERY SUGGESTIONS

After generating a dashboard, display suggested follow-up queries.

Example:

Suggested Analysis

• Compare revenue by customer segment
• Show top 5 campaigns
• Analyze monthly revenue trend

These suggestions must appear as clickable buttons.

When clicked, the dashboard updates accordingly.

Number of columns

Detected categorical fields

Dataset Summary

Columns: 6

Detected Metrics: Revenue
